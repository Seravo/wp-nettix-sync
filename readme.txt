WP Nettix Sync brings over data from nettix-sites like nettiauto.fi as custom post types.

== Description ==


== Installation ==

1. Make sure that you have a working datapipe from nettix.
2. Download and activate the plugin.
3. Define NETTIX_DEALERLIST or NETTIX_ADLIST in wp-admin
4. Installation done!
